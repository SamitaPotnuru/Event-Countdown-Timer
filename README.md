# Event-Countdown-Timer

A modern, responsive, and dynamic **event countdown timer** built with **HTML, CSS, and JavaScript**. Users can add multiple events, select an event, and see a live countdown with animated flipping digits.

## Features

- Add **one or more events** dynamically with name and date.  
- Select events from a **dropdown list** to switch countdowns.  
- Countdown displays **Days, Hours, Minutes, Seconds** horizontally.  
- **Animated flipping digits** for a modern and engaging look.  
- Fully **responsive** design for desktop and mobile devices.  
- Automatically updates if an **event changes**.  
- Shows a **celebration message** when the countdown ends.  

**Technologies Used**
1. HTML5
2. CSS3 
3. JavaScript
